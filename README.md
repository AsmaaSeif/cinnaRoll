## üåü Features & Objectives

- **Responsive Design**: Works beautifully on mobile, tablet, and desktop.  
- **Scroll Animations**:
  - **Cinnamon Images Movement**: As users scroll, the left and right cinnamon images move in opposite directions, creating a dynamic parallax effect.  
  - **Back-to-Top Button**: Appears when scrolling down, with smooth scrolling back to the top. The button changes color based on scroll position for visibility.  
  - Overall scrolling effects enhance user engagement and highlight key sections like About, Services, and Our Bits.  
- **Interactive Sections**:
  - **Home / Landing Section**: Attractive header with animated images.  
  - **About**: Homemade promise and story of CinnaRoll.  
  - **Our Service**: Pickup, Delivery, Same Day options.  
  - **Our Bits**: Showcase of different cinnamon roll products.  
- **Navigation Bar**: Responsive with hover effects.  
- **Footer**: Social media links with hover effects.
